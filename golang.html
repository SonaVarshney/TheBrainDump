🌀 Golang Notes

Quick notes and concepts I’ve learned while working with Go.

🌱 Basics

  • Go is statically typed, compiled, and super fast.
  • Has garbage collection and great concurrency support.
  • All Go code belongs inside a package.

🧵 Concurrency

  • goroutines are lightweight threads.
  • channels allow safe communication between goroutines.
  • Use select for non-blocking channel ops.

📦 Standard Library

  • net/http for building web servers.
  • fmt for formatting and printing.
  • os and io/ioutil for file handling.
“Don’t communicate by sharing memory, share memory by communicating.”
